Overview

ngx-zero-dialog is a lightweight, zero-dependency Angular library for modal dialogs built on the native HTML <dialog> API. It leverages browser capabilities for focus trapping, backdrop, accessibility, and stacking context — no CDK or overlay system needed.

Key features:

  • Open dialogs with a Component or TemplateRef
  • Built-in default host — or bring your own for consistent dialog chrome
  • Observable-based API with typed results
  • beforeClose guard to intercept and prevent closing
  • Pure CSS enter/leave animations via class toggling
  • Configurable backdrop click and Escape key behavior

When exploring the demo examples, open DevTools to observe how each dialog is reflected in the DOM.

Requires Angular 16+
