Custom animations

ngx-zero-dialog uses CSS class toggling for enter/leave animations. Two classes are applied to the <dialog> element:

  • .ngx-zero-dialog-hidden — starting state (before open transition)
  • .ngx-zero-dialog-visible — added after showModal(), removed on close
